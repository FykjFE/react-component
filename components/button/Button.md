Button 按钮:

**按钮颜色**

```jsx

<Button type={'primary'}>primary</Button>
<Button>测试</Button>
```

**按钮大小**

```jsx
<Button size={'small'} type={'default'}>测试</Button>
<Button type={'default'}>测试</Button>
<Button size={'small'} type={'default'}>测试</Button>
```
**按钮状态**

```jsx
<Button type={'default'}>正常</Button>
<Button disabled type={'default'}>禁用</Button>
```
